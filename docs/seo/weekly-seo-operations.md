# Weekly SEO Operations

Run this checklist every week (same weekday).

## 1) Search Console review

- Compare week-over-week:
  - Impressions
  - Clicks
  - Average CTR
  - Average position
- Filter by pages:
  - `/aviation-compliance-audit-services`
  - `/aviation-quality-software`
  - `/faa-repair-station-audit-checklist`
  - `/as9100-internal-audit-software`
  - `/aviation-audit-readiness`

## 2) Query optimization

- For pages with position 8-20:
  - strengthen H2 copy around matching query terms
  - add one FAQ or proof section that aligns to user intent
- For pages with good position but low CTR:
  - rewrite title and meta description for stronger specificity

## 3) Internal linking pass

- Add links from support articles to primary conversion pages.
- Ensure anchor text stays descriptive and not repetitive.

## 4) Indexation checks

- Confirm sitemap remains valid.
- Spot-check 2 pages with URL inspection.
- Resolve coverage warnings.

## 5) Reporting output

Capture a one-page weekly report:

- Top 5 query gains
- Top 3 page gains
- New backlinks earned
- Actions taken this week
- Next week plan
