# Help Image Assets

Store in-app Help Center screenshots and supporting visuals here.

## Naming

- Use: `<module>-step-<nn>-<state>.png`
- Example: `paperwork-review-step-02-start-review.png`

## Guidelines

- Capture one clear moment per image (entry, key action, result).
- Keep UI labels readable.
- Prefer consistent viewport and similar crop framing across a workflow set.
- Pair every image with concise alt text in markdown.
